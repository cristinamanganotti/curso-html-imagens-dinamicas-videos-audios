#  Projeto: Manipulação de Mídias em HTML5

Este projeto foca na implementação técnica de áudios, vídeos e imagens dinâmicas, priorizando a organização de diretórios e a experiência do usuário.

## Desafios Superados
O maior desafio foi a estruturação da tag `<audio>`. Aprendi a:
- Organizar mídias em pastas específicas para projetos escaláveis.
- Implementar controles nativos e pré-carregamento de metadados para performance.
- Criar alternativas de download para garantir acessibilidade em navegadores antigos.

## Tecnologias Utilizadas
- **HTML5:** Tags semânticas, `<picture>` e `<audio>`.
- **Git/GitHub:** Controle de versão e documentação.
